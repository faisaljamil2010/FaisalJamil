  • A Gamified Approach for Optimal Waste Management based on IoT-Blockchain Network using Deep Reinforcement Learning

    Sponsor: European Research Consortium for Informatics and Mathematics

    This project models an optimal waste management system using a gamified approach that leverages multi-agent deep reinforcement learning (DRL). The system monitors waste levels in bins, overflow status, and executes actions like waste collection, disposal, or no action. The goal is to maximize rewards and minimize penalties for agent actions, improving waste management efficiency. The approach is modeled using Markov Decision Processes (MDP), with strategies and algorithms optimized for real-time decision-making. The gamification aspect encourages citizen participation by awarding points for proper waste disposal, while municipal workers use the system to avoid waste overflow.

    Skills: Deep Reinforcement Learning, Multi-Agent Systems, IoT, Blockchain, Markov Decision Processes (MDP), Python, Data Analysis, Gamification

  • Toward Intelligent Open-Ended Question Evaluation based on Predictive Optimization

    Sponsor: European Research Consortium for Informatics and Mathematics

    This system develops an intelligent student evaluation model based on a predictive optimization approach, analyzing question structure, necessary keywords, language, and conceptual aspects. A deep neural network is used as a learning module, with particle swarm optimization and gradient descent for optimizing the weighting parameters. It employs natural language processing techniques for evaluating answer relevance, using the WordNet library and Growbag dataset to compute a semantic similarity score. The platform is validated using NTNU student exam data, and performance metrics include precision, recall, and f-measure.

    Skills: Deep Neural Networks, Predictive Optimization, Particle Swarm Optimization, NLP, WordNet, Python, Data Analysis, Research Writing
